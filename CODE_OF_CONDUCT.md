# Código de Conducta

Este proyecto adopta el **Contributor Covenant v2.1**.

Nos comprometemos a ofrecer un entorno acogedor y libre de acoso para todas las personas, sin importar
edad, identidad o expresión de género, discapacidad, etnia, nivel de experiencia, nacionalidad, apariencia,
religión u orientación. Esperamos respeto, empatía y buena fe en todas las interacciones.

No se tolera el acoso, el lenguaje o imágenes ofensivas, los ataques personales ni la divulgación de
información privada de terceros.

Para reportar conductas inaceptables, contacta a los mantenedores en `<correo de contacto>`. Los reportes
se revisarán de forma confidencial.

Texto completo del Contributor Covenant: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
